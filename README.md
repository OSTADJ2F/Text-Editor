# Simple Text Editor

A lightweight text editor built with Python and Tkinter. This application provides basic text editing functionality with a simple graphical user interface.

## Features

- Create new text files
- Open existing text files
- Save files
- Save files with a new name
- Simple and intuitive interface
- Fixed window size for consistent experience

## Requirements

- Python 3.x
- Tkinter (usually comes with Python installation)

## Usage

1. Run the application:
```bash
python textEditor.py
```

2. Use the File menu to:
   - Create new files
   - Open existing files
   - Save current file
   - Save file with a new name
   - Quit the application

## Window Specifications

- Width: 400 pixels
- Height: 400 pixels
- Fixed window size

## Development

This project uses:
- `tkinter` for the GUI
- Built-in Python file handling for read/write operations
